# Google Drive API

## Prerequisites

- Node.js (with npm)
- Google Cloud Project with Drive API enabled
- OAuth 2.0 credentials (Client ID)

## Setup

1. Copy the .env.example file to .env and fill in your Google Client ID.

   ```bash
   cp .env.example .env
   ```
   
2. Install dependencies:

   ```bash
    npm install
    ```
   
3. Start the server:

   ```bash
    npm run dev
    ```
